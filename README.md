# Zapret UI

Графический интерфейс для [Zapret](https://github.com/bol-van/zapret) — обход блокировок Discord, YouTube и других сервисов.

## Установка

1. Скачайте **ZapretUI-Setup.exe** из [Releases](https://github.com/RaccoonLaptop/ZapretUI/releases)
2. Запустите установщик
3. Откройте программу → выберите стратегию → **Запустить**

Всё остальное (скачивание zapret, обновления) программа делает сама.

## Сборка

```powershell
.\build-installer.ps1
```

Результат: `ZapretUI-dist\ZapretUI-Setup.exe`

## Требования

- Windows 10/11 (64-bit)
- Права администратора (запрашиваются автоматически)
