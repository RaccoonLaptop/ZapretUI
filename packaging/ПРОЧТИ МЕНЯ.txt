  ZAPRET UI — Автор: Niko
  ========================

  УСТАНОВКА (рекомендуется)
  -------------------------
  1. Скачайте ZapretUI-Setup.exe с GitHub Releases
  2. Запустите установщик
  3. Укажите папку zapret-discord-youtube (где service.bat и bin\)
  4. Готово — ярлык появится в меню Пуск

  Если zapret ещё нет — скачайте с:
  https://github.com/Flowseal/zapret-discord-youtube

  ЗАПУСК
  ------
  Меню Пуск → Zapret UI
  или ярлык на рабочем столе

  ТРЕБОВАНИЯ
  ----------
  - Windows 10/11 (64-bit)
  - Папка zapret-discord-youtube (Flowseal)

  УДАЛЕНИЕ
  --------
  Параметры Windows → Приложения → Zapret UI → Удалить
