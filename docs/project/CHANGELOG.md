# История изменений Zapret UI

Полный журнал версий. Актуальный релиз: **v1.6.8**.

Формат: версия → основные изменения (по коммитам и релизам GitHub).

---

## v1.6.8 (2026-05-27)

- Меню трея на WPF со стилем **NavButton** (подсветка при наведении, как в боковом меню)
- Замена WinForms ContextMenuStrip на `TrayMenuPopup`

## v1.6.7 (2026-05-27)

- Исправлена видимость текста статуса в меню трея (светлый текст на тёмном фоне)

## v1.6.6 (2026-05-27)

- Исправлено пустое меню трея (текст и кнопки не отображались)

## v1.6.5 (2026-05-27)

- Меню трея: запуск/остановка обхода, статус, зелёная иконка при работе
- Тёмный дизайн меню

## v1.6.4 (2026-05-27)

- Исправлена ложная ошибка «winws не запустился» после остановки во время запуска
- Исправлен двойной вызов `ui-bridge.ps1` при старте стратегии

## v1.6.3 (2026-05-27)

- Исправлен зависание при старте (deadlock при применении дефолтов на UI-потоке)
- Дефолты после установки применяются в фоне после показа окна

## v1.6.2 (2026-05-27)

- После чистой установки: Game Filter TCP+UDP, IPSet loaded

## v1.6.1 (2026-05-27)

- Исправлена загрузка локализации (JSON в папке `Resources/`)

## v1.6.0 (2026-05-27)

- Английский язык интерфейса (RU/EN в «Сервис»)

## v1.5.4

- Тематические диалоги подтверждения и информации
- Улучшения вкладки «Сервис» (цвета статусов, ссылки)

## v1.5.3

- Фон **Wavy** по умолчанию после установки

## v1.5.2

- Исправлено зависание «Запускается…» после успешного старта winws

## v1.5.1

- Индикатор прогресса при запуске стратегии

## v1.5.0

- Остановка winws перед выходом из программы

## v1.4.9

- Корректный разбор аргументов winws из .bat при скрытом запуске

## v1.4.8

- Скрытая консоль при запуске стратегии из UI

## v1.4.6 / v1.4.5

- Встроенная стратегия **Niko_ALT11**
- Окно прогресса при установке обновления

## v1.4.4 — v1.4.2

- Обновление только через `ZapretUI-Setup.exe`
- Исправления диалога поиска/замены, иконки в панели задач

## v1.4.0 — v1.3.x

- Анимированные фоны главной (Wavy, Aurora, Meteors и др.)
- Скорость анимации, затем фиксированная 15%
- Списки внутри стратегий
- Полноэкранный фон с полупрозрачной оболочкой
- Удаление конфигов/списков, поиск в редакторе

## v1.2.x

- Выбор фона главной (bedolaga-cabinet style)
- Двухшаговое обновление приложения
- Исправления title bar, иконки, первого запуска безопасности
- Звёздный фон, проверка обновлений Flowseal при старте

## v1.1.x

- Установщик Inno Setup с автоскачиванием Flowseal
- Редактор стратегий, цветная консоль, вкладки Сервис/Стратегии
- Исправления крашей, UAC elevation, полная замена при обновлении

## v1.0.0

- Первый релиз: графический интерфейс для конфигов Flowseal zapret
