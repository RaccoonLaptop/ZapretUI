Быстрый старт

  1. Скачайте и установите ZapretUI-Setup.exe.
  2. При первом запуске дождитесь загрузки компонентов Flowseal.
  3. На вкладке Главная выберите стратегию и нажмите ЗАПУСТИТЬ.

Системный трей

Зелёная иконка — обход работает.

Синяя иконка — остановлено.

ПКМ по иконке: статус, запуск/остановка, открыть программу, выход.

После установки по умолчанию

  • Game Filter: TCP + UDP
  • IPSet: загружен
  • Фон главной: Wavy

Решение проблем

  • winws не запустился — запуск от администратора, другая стратегия.
  • Не открывается окно — установите последнюю версию с GitHub.
  • Сайты не работают — обновите Flowseal в «Сервис», смените стратегию.

Документация для разработчиков

В репозитории: папка docs/project/, файл AGENTS.md — полный контекст для Cursor AI.

История изменений

Quick start

  1. Download and install ZapretUI-Setup.exe.
  2. On first launch wait for Flowseal components to download.
  3. On Home pick a strategy and press START.

App sections

TabDescription
HomeStart/stop bypass, strategy picker, background
Strategies.bat editor, lists, copy configs
ServiceGame filter, IPSet, language, updates, security
ConsoleLogs and diagnostics

System tray

Green icon — bypass running.

Blue icon — stopped.

Right-click: status, start/stop, open app, exit.

Fresh install defaults

  • Game Filter: TCP + UDP
  • IPSet: loaded
  • Home background: Wavy

Troubleshooting

  • winws did not start — run as administrator, try another strategy.
  • Window won’t open — install the latest release from GitHub.
  • Sites still blocked — update Flowseal in Service, change strategy.
