# Руководство пользователя Zapret UI

> Онлайн-версия: https://raccoonlaptop.github.io/ZapretUI/

## Установка

1. Скачайте **ZapretUI-Setup.exe** с [Releases](https://github.com/RaccoonLaptop/ZapretUI/releases).
2. Запустите установщик, при необходимости подтвердите UAC.
3. При первом запуске программа скачает компоненты Flowseal (один раз, нужен интернет).

Программа устанавливается в `%LocalAppData%\ZapretUI`. Конфиги zapret — в `%LocalAppData%\ZapretUI\zapret`.

## Главная — запуск обхода

1. Откройте вкладку **Главная**.
2. Выберите стратегию в списке (например `general.bat`).
3. Нажмите **ЗАПУСТИТЬ**.
4. Статус сменится на **Работает** (зелёная точка).
5. Для остановки — **ОСТАНОВИТЬ**.

Переключатель фона (✦) меняет анимацию на главной. По умолчанию — **Wavy**.

## Стратегии

- Просмотр и редактирование `.bat` с подсветкой синтаксиса.
- **Запустить** — запуск выбранной стратегии.
- Создание копии, переименование, удаление (служебные `service*.bat` защищены).
- **Списки** — редактирование `lists/*.txt` для выбранного конфига.
- Поиск и замена в редакторе.

## Сервис

| Раздел | Назначение |
|--------|------------|
| Game Filter | Фильтр портов для игр: TCP+UDP / только TCP / только UDP / выкл |
| IPSet Filter | Загрузка списка IP из Flowseal |
| Авто-обновление | Проверка новых версий при старте |
| Язык | Русский / English (нужен перезапуск) |
| Безопасность | Исключения Windows Defender и правила брандмауэра |
| Обновления | Zapret UI и Flowseal |
| Сброс сети | `netsh` + сброс DNS (нужен админ) |

После **чистой установки** по умолчанию: Game Filter **TCP+UDP**, IPSet **загружен**.

## Консоль

Окно логов: вывод PowerShell и сообщений программы. Открывается из меню **Консоль**.

## Системный трей

Иконка Zapret UI всегда в области уведомлений.

- **ПКМ** — меню: статус, запуск/остановка, открыть программу, выход.
- При работе обхода кольцо иконки **зелёное**.
- **Двойной клик** — открыть окно программы.

Если обход запущен и вы закрываете окно крестиком — программа сворачивается в трей (обход не останавливается).

## Обновление программы

При наличии новой версии появится запрос. Скачивается `ZapretUI-Setup.exe`, установка с прогрессом, затем перезапуск.

## Требования и права

- Windows 10/11 x64
- Права администратора (запрашиваются автоматически)
- Интернет для первой загрузки Flowseal и обновлений

## Решение проблем

| Проблема | Что сделать |
|----------|-------------|
| «winws не запустился» | Запустите Zapret UI от имени администратора; попробуйте другую стратегию |
| Ошибка после остановки | Обновитесь до v1.6.4+ |
| Программа не открывается | Удалите `%LocalAppData%\ZapretUI\utils\.zapretui_defaults_v1` и перезапустите; установите последний релиз |
| Discord/YouTube не работают | Другая стратегия; обновите Flowseal в «Сервис» |
