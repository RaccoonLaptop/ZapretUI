  ZAPRET UI — Автор: Niko
  ========================

  УСТАНОВКА
  ---------
  1. Скачайте ZapretUI-Setup.exe
  2. Запустите — всё остальное программа сделает сама:
     - установит Zapret UI
     - скачает zapret (Flowseal) с GitHub
  3. Готово — ярлык в меню Пуск

  Ничего выбирать и распаковывать не нужно.

  ЗАПУСК
  ------
  Меню Пуск → Zapret UI

  ТРЕБОВАНИЯ
  ----------
  - Windows 10/11 (64-bit)
  - Интернет (для первой установки компонентов zapret)

  УДАЛЕНИЕ
  --------
  Параметры Windows → Приложения → Zapret UI → Удалить
